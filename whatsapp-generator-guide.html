🎯Pengenalan

WhatsApp Message Generator adalah tool untuk membuat pesan undangan pernikahan yang dipersonalisasi secara otomatis. Setiap tamu akan mendapatkan:

  • Pesan undangan dengan nama mereka
  • URL undangan digital yang personal
  • Format profesional dan konsisten

Kenapa Menggunakan Tool Ini?

⏱️ Hemat Waktu

Generate ratusan pesan dalam hitungan detik

👤 Personal

Setiap tamu mendapat URL dengan namanya

💼 Profesional

Template yang sudah didesain dengan baik

📊 Tracking

URL berbeda memudahkan tracking siapa yang buka undangan

🚀Cara Menggunakan

Langkah 1: Buka Tool

  1. Buka Link https://raz.my.id/iqbalshania/whatsapp-generator.html dengan browser
  2. Pastikan koneksi internet aktif (opsional, tool tetap bisa digunakan offline)

Langkah 2: Pilih Template

Klik salah satu dari 4 template yang tersedia:

  • Formal - Untuk keluarga besar, tamu VIP
  • Santai - Untuk teman dekat
  • Singkat - Untuk broadcast cepat
  • Custom - Edit sendiri sesuai keinginan

Langkah 3: Isi Data

Base URL sudah otomatis terisi: https://raz.my.id/iqbalshania/index.html
Bisa diubah jika deploy di domain lain

Nama Tamu - Ketik nama tamu, contoh:

  • Bapak Ahmad dan Keluarga
  • Ibu Siti
  • Dr. Budi Santoso, M.Sc.

Langkah 4: Generate

  1. Klik tombol "Generate WhatsApp Message"
  2. Preview pesan akan muncul di bawah
  3. URL personal juga akan ditampilkan

Langkah 5: Kirim

Pilih salah satu cara:

  • 📋 Copy Pesan - Copy pesan, paste ke WhatsApp manual
  • 🔗 Copy URL - Copy hanya URL-nya
  • 💬 Buka WhatsApp - Langsung buka WhatsApp Web dengan pesan
  • 💾 Download TXT - Simpan sebagai file text

⭐Fitur-Fitur

1. Single Message Generator

Generate pesan untuk 1 tamu per kali.

Kapan digunakan:
  • Tamu VIP yang perlu pesan khusus
  • Testing template sebelum bulk send
  • Tamu yang baru ditambahkan

2. Bulk Message Generator

Generate pesan untuk banyak tamu sekaligus.

Cara pakai:
  1. Scroll ke bagian "Bulk Message Generator"
  2. Masukkan daftar nama (1 nama per baris)
  3. Klik "Generate Bulk Messages"
  4. Download hasil dalam format CSV atau TXT

3. Statistics

Monitor berapa pesan yang sudah dibuat:

  • Total pesan yang di-generate
  • Jumlah karakter
  • Panjang URL

🔧Troubleshooting

Problem 1: URL Terlalu Panjang

Solusi:
  • Gunakan URL shortener (bit.ly, s.id)
  • Deploy di domain pendek
  • Jangan pakai nama yang terlalu panjang

Problem 2: Nama dengan Karakter Khusus

Contoh: H. Ahmad & Ibu Siti

Solusi: Tool otomatis encode karakter khusus. & menjadi %26, Spasi menjadi %20. Tetap aman digunakan ✅

Problem 3: Template Hilang

Solusi:
  1. Refresh halaman (F5)
  2. Template akan auto-load saat halaman dibuka
  3. Klik tombol template untuk ganti

Problem 4: Copy Tidak Berfungsi

Solusi:
  • Pastikan browser mendukung Clipboard API
  • Gunakan browser modern (Chrome, Edge, Firefox)
  • Atau klik manual text → Ctrl+C
